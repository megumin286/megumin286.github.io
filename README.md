## ⚠️ 重要说明

这是一个**纯静态网站**版本，可以直接上传到GitHub Pages或其他静态托管服务。

- ✅ 无需数据库
- ✅ 无需后端服务器
- ✅ 所有浏览器可直接运行
- ✅ 数据存储在浏览器localStorage中

## 🚀 快速开始

### 方法1：直接打开
双击 `index.html` 文件即可在浏览器中打开

### 方法2：使用本地服务器
```bash
# 简单方式
python -m http.server 8888

# 或使用Node.js
npx serve .

# 或使用PHP
php -S localhost:8888
```

然后访问：http://localhost:8888

## 📁 项目文件

```
helldivers-recruitment/
├── index.html          # 主页面
├── success.html        # 报名成功页面
├── styles.css          # 样式文件
├── script.js           # 前端逻辑
├── README.md           # 项目说明
├── recruitment_poster.jpg # 征兵海报（需自行准备）
└── propaganda.mp4      # 宣传视频（需自行准备）
```

## 🎬 添加媒体文件

### 1. 征兵海报
将图片重命名为 `recruitment_poster.jpg` 放在项目目录

### 2. 宣传视频
将视频重命名为 `propaganda.mp4` 放在项目目录

## 🌐 部署到GitHub Pages

1. 创建GitHub仓库
2. 上传所有文件
3. 进入 Settings → Pages
4. 选择 `main` 分支和 `/(root)` 目录
5. 保存后等待部署完成
6. 访问 `https://你的用户名.github.io/仓库名`

## 📱 使用说明

### 作为访问者
1. 访问网站首页
2. 观看自动播放的宣传视频
3. 查看福利信息（点击卡片展开）
4. 填写并提交报名表
5. 完成报名！

### 作为管理者
1. 访问网站
2. 点击页面底部的「管理入口」按钮
3. 查看所有报名数据和统计信息
4. 可以刷新数据、导出数据或清除数据

## ⚠️ 注意事项

- 宣传视频 `propaganda.mp4` 需要自行准备并放置在项目根目录
- 海报图片 `recruitment_poster.jpg` 需要自行准备
- 所有相对路径资源都在同一目录下
- 数据存储在浏览器localStorage中（不同浏览器数据不互通）
- 清除浏览器缓存会导致报名数据丢失
